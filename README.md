# World Cup CN Tracker

一个面向中文读者的世界杯赛事追踪网页，按北京时间展示赛程、比分、小组积分、出线趋势、淘汰赛路径和预测分析。

## 在线访问

https://soling0410.github.io/world-cup-cn-tracker-2026/

## 可以查看什么

- 今日赛程和按日期切换的比赛列表
- 全部赛程
- 已结束比赛比分
- 未开始比赛的北京时间开球时间
- 小组积分、出线形势和趋势
- 淘汰赛对阵路径
- Elo + 蒙特卡洛模型生成的预测分析
- 数据来源和最后更新时间

## 数据来源

核心赛程、比分、比赛状态和对阵信息来自 FIFA 官方接口：

https://api.fifa.com/api/v3/calendar/matches?language=en&count=500&idSeason=285023

页面中的比赛时间统一转换为北京时间，并按北京时间自然日分组。第三方媒体链接只用于交叉核验或补充阅读，不作为主数据源。

## 数据准确性

项目以 FIFA 官方数据为准。未结束比赛只显示北京时间开球时间，不提前改写比分；已结束比赛显示官方同步到的比分。

如果官方接口尚未更新某场赛果，页面也会保持待更新状态。

## 预测分析说明

预测页使用 Elo 强度估计和蒙特卡洛模拟生成趋势概率。

预测内容仅供赛事观察和讨论参考，不构成确定性判断，也不应用于投注或其他高风险决策。

## 更新机制

页面通过 GitHub Actions 定时同步 FIFA 官方数据并发布到 GitHub Pages。由于 GitHub Actions 的定时任务可能存在延迟，页面以站内显示的最后更新时间为准。
